# SOA Health Services
Desglose de servicios dentro de un **sistema de salud simulado**, Todas las funcionalidades definidas por el equipo de desarrollo, tomando como referencias diversas fuentes.
 
### Laravel Setup
```
composer install
npm install
cp .env.example .env
```

### Views Setup
```
npm install
```

### Inicializar servidor
```
php artisan serve
```
